# 🎴 직장인 타로 웹앱

당신의 커리어를 위한 78장의 답

## 🌟 특징

- ✅ 78장 타로 카드 (메이저 22장 + 마이너 56장)
- ✅ 직장인 맞춤 해석
- ✅ 원카드 / 쓰리카드 스프레드
- ✅ 모바일 최적화
- ✅ PWA 지원 (앱처럼 설치 가능)
- ✅ 완전 무료

## 🚀 사용 방법

1. 스프레드 선택 (원카드 또는 쓰리카드)
2. 카드 클릭해서 뽑기
3. 해석 확인
4. 전체 카드 보기에서 78장 모두 탐색 가능

## 💻 로컬 실행

1. 파일 다운로드
2. `index.html` 파일을 브라우저로 열기
3. 끝!

## 🌐 GitHub Pages 배포

1. GitHub 저장소 생성
2. 파일 업로드
3. Settings > Pages에서 배포
4. `https://yourusername.github.io/office-tarot` 접속

## 📱 모바일 앱처럼 사용하기

### iOS (Safari):
1. 웹사이트 접속
2. 공유 버튼 클릭
3. "홈 화면에 추가"
4. 앱처럼 사용!

### Android (Chrome):
1. 웹사이트 접속
2. 메뉴 (⋮) 클릭
3. "홈 화면에 추가"
4. 앱처럼 사용!
